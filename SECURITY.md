# Security Policy

## Reporting a vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, email **robertgumeny7@gmail.com** with:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept (if available)
- Any relevant file samples or test cases

You will receive a response within 72 hours. Please allow time to assess the report and prepare a fix before public disclosure.
